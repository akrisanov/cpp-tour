# Prelude

- C++ today allows to express ideas more clearly, more simply, and more directly than C++98 or C++11.
- The resulting programs are better checked by the compiler and run faster.
- [ISO C++](https://isocpp.org/) is the organization that standardizes C++ and provides resources for learning and using C++.
- CppCon and Meeting C++ are two conferences where you can learn about the latest developments in C++.
- [CppReference](https://en.cppreference.com/w/) is a great resource for learning about C++ features and libraries.
- [C++ Core Guidelines](https://isocpp.github.io/CppCoreGuidelines/CppCoreGuidelines) is a set of guidelines for writing modern C++ code.
