## Structures

- Types built into C++ include:
  - fundamental types (e.g., int, double, char)
  - compound types (e.g., arrays, pointers)
  - user-defined types (e.g., structures)
- User-defined types are refered to as classes and enumerations.
- The `new` operator allocates memory from the free store (also known as the heap) and returns
  a pointer to the allocated memory.
- Objects allocated on the free store are independent of the scope in which they were created and
  remain alive until they are explicitly deallocated using the `delete` operator.
- The standard library types have lower-case names, while user-defined types have upper-case names.
- To access members of a structure, we use the dot operator (`.`) for objects and
  the arrow operator (`->`) for pointers to structures.

## Classes

- To keep the data and the operations that manipulate that data together, we can define a class.
- A class is a user-defined type that consists of members, which can be data, functions, or type members.
- The interface of a class is defined by its public members, while the implementation is defined by its private members.
- Conventionally:
  - class names start with an uppercase letter (e.g., `Person`, `Car`)
  - member variables (data members) start with a lowercase letter (e.g., `name`, `age`)
  - member functions (methods) also start with a lowercase letter (e.g., `getName()`, `setAge()`)
  - constructors are special member functions that initialize objects of a class and have the same name as the class
  - public declarations come before private declarations in a class definition
